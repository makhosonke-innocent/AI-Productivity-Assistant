# AI Workplace Productivity Assistant

## 📌 Problem Statement
Professionals waste 10+ hours/week on repetitive tasks like drafting emails, summarizing meetings, planning schedules, and researching. This AI assistant automates these tasks to save time and improve productivity.

## 🎯 Solution Overview
A Streamlit-based web app with 5 AI-powered features:
1. **Smart Email Generator** – adapts tone & audience
2. **Meeting Summarizer** – extracts actions & decisions  
3. **Task Planner** – prioritizes & schedules tasks
4. **Research Assistant** – simplifies complex topics
5. **Chatbot Interface** – interactive Q&A

## 🛠️ Tools Used
- **Frontend**: Streamlit
- **AI Model**: OpenAI GPT-3.5 Turbo
- **Language**: Python 3.9+
- **Version Control**: GitHub
